Zona de administración — Acciones irreversibles
Las acciones de esta sección eliminan datos permanentemente de los archivos Excel. No hay forma de deshacer. Se recomienda hacer un respaldo antes de continuar.
Registros Transaccionales (movimientos, avances, préstamos)

Borra los registros históricos. Los catálogos (materiales, herramientas, etc.) se conservan. Útil para iniciar un nuevo período.

Catálogos (materiales, herramientas, empleados…)

Borra los catálogos maestros. Todos los registros que referencian estos datos quedarán sin referencia válida.

Respaldo del Sistema — backup y restauración de datos
Exportar backup

Descarga un archivo .zip con todos los datos del sistema (catálogos y registros). Guárdalo en un lugar seguro antes de realizar cualquier operación de borrado.

Restaurar backup

Carga un .zip exportado previamente. Los archivos Excel actuales serán sobreescritos con el contenido del backup.

Generar paquete del servidor — ZIP para desplegar en PC casa (Ubuntu + Cloudflare)

Genera un archivo .zip con todo lo necesario para instalar el sistema en tu PC servidor en casa (Ubuntu). Copia el ZIP al servidor y ejecuta sudo bash update_server.sh nombre.zip. Incluye:

  • Aplicación Flask para producción (app.py) + capa de datos (db.py)
  • Base de datos SQLite con todos tus datos actuales migrados (taller.db)
  • Scripts de instalación y actualización: setup_ubuntu.sh, update_server.sh
  • Servicio del sistema: taller.service (inicio automático con Ubuntu)
  • Plantillas HTML, archivos estáticos y manual de usuario
  • Guía completa README_SERVIDOR.md (Ubuntu + Cloudflare Tunnel)
Primera vez: ejecuta sudo bash setup_ubuntu.sh antes de update_server.sh. Consulta README_SERVIDOR.md para la configuración completa de Cloudflare Tunnel.
